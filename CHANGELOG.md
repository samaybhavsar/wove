# Changelog

All notable changes to Wove are documented here. The format follows
[Keep a Changelog](https://keepachangelog.com/), and the project uses
[semantic versioning](https://semver.org/).

## [1.0.0] — 2026-06-11

### Added
- Initial release of **Wove**, a minimal, typographic block theme (Full Site Editing, `theme.json` v3) for a personal site and blog.
- Templates: front page (intro + year-led "Writing" index), blog index, single post, page, archive, search, 404.
- Self-hosted **Newsreader** serif with fluid type; monochrome palette.
- **Automatic dark mode** (respects OS preference) with a header toggle and no flash of the wrong theme.
- **One-click setup**: starter content plus a first-run admin notice that creates the Home/Blog/About/Contact pages and configures the front page.
- **Appearance → Wove** settings page for email and social links — the single source for the footer, the Contact block, and the SEO `sameAs` data.
- Single-post niceties: categories, estimated reading time, polished long-form styling (lists, tables, images, captions, inline code), prev/next navigation, comments.
- Built-in lightweight SEO: meta description, Open Graph, Twitter cards, JSON-LD.
- Accessibility: skip link, visible focus indicators, sequential headings, `prefers-reduced-motion` support, AA-contrast palette.
- Translation-ready (`/languages/wove.pot`).
